Build a polished mobile-first responsive Korean web app called “InnerMeter”.

==================================================
PROJECT OVERVIEW
==================================================

Create a Korean-first personality/relationship/fun/tarot test platform called InnerMeter.

Brand:
- Name: InnerMeter
- Tagline: 당신의 내면을 가볍고 재미있게 측정해보세요

This is a mobile-first responsive web app, not a fixed-width fake phone layout.
It must feel modern, trendy, premium, and highly shareable.

Core goals:
1. Provide multiple personality-style tests
2. Provide Korean-first UX
3. Support randomized test flow
4. Generate beautiful shareable result images automatically
5. Maximize retention through recommended next tests
6. Prepare structure for future multilingual support
7. Do not add ads yet
8. Do not add login/payment/backend complexity unless absolutely necessary

AUTO RESULT IMAGE GENERATION
==================================================

This is critical.

Create an automatic shareable result image generation system for each test result.

Purpose:
Users should be able to generate a beautiful result card image and share it to SNS.

Requirements:
1. Generate a visually polished result image on the frontend
2. Result image must include:
   - InnerMeter logo/brand name
   - test title
   - result title
   - short summary or short phrase
   - subtle decorative background
3. The image must look premium and trendy, not plain
4. It must feel good enough to post to social media
5. Korean typography must look balanced
6. Design should work well on mobile
7. Provide button(s):
   - 결과 이미지 저장
   - 이미지로 공유하기 (if feasible)
   - 링크 복사
8. If native sharing is available on mobile, support it
9. Use a clean frontend approach such as HTML-to-image rendering or canvas export
10. Make sure the generated image matches the actual result data

Result image style direction:
- clean premium card
- gradient or soft-tone background
- rounded corners
- modern typography
- subtle decorative shapes or glow
- distinct visual variants by category or result type
- not too noisy
- elegant enough for Instagram story / X / Kakao share feeling

Suggested image dimensions:
- portrait-friendly share card
- e.g. 1080x1350 or similar ratio
- also ensure it scales well for mobile preview

Important:
- Do not make the image look generic
- Do not make it plain white text on flat background
- Design matters a lot here
- The image should feel intentionally designed

Also:
- create a reusable ResultShareCard component
- allow the result page to preview this share image before saving/sharing
